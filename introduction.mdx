## What is Voxora?

Voxora is an open-source, developer-friendly live chat and voice support solution built for modern web applications. It provides a complete customer support platform with realtime messaging, team management, and an embeddable chat widget.

## Monorepo Structure

Voxora is organized as a monorepo with the following components:

- **apps/web**: Next.js 15 with React 19 (App Router, Tailwind CSS)
- **apps/api**: Express 5 + Socket.IO backend with MongoDB and Redis
- **packages/***: Shared ESLint and TypeScript configurations
- **docker/**: Local development services (MongoDB, Redis, Mongo Express, MailHog)

## Who is Voxora for?

Voxora is perfect for:

- **Startups** looking for a self-hosted customer support solution
- **Developers** who want full control over their support infrastructure
- **Teams** that need realtime collaboration features
- **Businesses** seeking an open-source alternative to proprietary chat platforms
